<b>Definition:</b> This is an activity an <b>Agent</b> participates in as a custodian of a <b>CollectionObject</b>.<br/><b>Scope: </b>It may contain data about long term responsibility of a <b>CollectionObject</b>. It could also contain data about changes to the curation of an object.<br/><b>Context:</b> Multiple <b>Agents </b>can be associated with a single <b>AgentRole</b>, but some activities can only have one related <b>Agent</b>.<br/>This is the based on the <i>Activity </i>in the PROV 3-axiom schema. (with the <b>CollectionObject</b> being the <i>used Entity</i> and the <b>Agent </b>the <i>wasAssociatedWith Agent</i>)<br/><b>Examples:</b> See Role attribute.<br/>
