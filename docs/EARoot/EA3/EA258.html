The <font color="#0000ff"><i>RecordType </i></font>field allows you to tell EMu what type of catalogue record you are about to enter. Selecting an option here with switch on or off tabs as appropriate. If you think there is a tab you should be seeing speak to Botany Sys-Admin.<br/>The options are: <br/><b>GENERAL OPTIONS</b><br/>1) <font color="#008000"><b>Specimen - single</b></font>; Use this for specimens which CAN have a barcode each. Eg sheets with 1 or more specimens. <br/>2) <font color="#008000"><b>Specimen - multiple</b></font>; Use this option for specimens which CANNOT have a barcode each. For example a microscope slide with many specimens on.<br/>3) <font color="#008000"><b>Bound Volume</b></font>; Use this option to database a Herbarium volume such as a Sloane volume.<br/>4) <font color="#008000"><b>Bound Volume Page</b></font>; Use this option to database the individual page in a Bound volume, not for the specimen itself. You can then attach these the the bound volume record.<br/>5) <font color="#008000"><b>Field Notebook (Single Page)</b></font>; Use this option to database a whole page from a collection note boot.<br/>6) <font color="#008000"><b>Field Notebook (Double Page)</b></font>; Use this option to database a notebook entry which cross an open page. Eg:<br/><br/>7) <font color="#008000"><b>Image (non-digital)</b></font>; Use this option to database photographic slides and photographs which are NOT barcoded specimens. <br/>8) <font color="#008000"><b>Image (digital)</b></font>; Use this option to database images which ONLY exist in electronic format for example Scans of live cultures.<br/>9) <font color="#008000"><b>Transient Lot</b></font>; Use this option ONLY if you are databasing a group of objects which will only remain together for a finite length of time. For example a folder of specimens removed for freezing. It may also be used to database Backlog boxes and large loans if there is not time to database individual specimens.<br/>10) <font color="#008000"><b>Artefact</b></font>; If none of the other object types works for you, then use this generic artefact option or ask me!<br/><b>MOLECULAR OPTIONS</b><br/>11) <font color="#008000"><b>Silica Gel Specimen</b></font>;
